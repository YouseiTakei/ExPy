Welcome to ExPy's documentation!
================================

ExPy is a Python library for scientific computing.

It contains among other things:
 - sophisticated functions
 - useful rendering formula, Formula deformation.
 - simply converting Jupyter notebook to PDF.

ExPy is licensed under the Mit license, enabling reuse with few restrictions.


Getting Started
===============

To install ExPy, we strongly recommend using a scientific Python distribution.
See Installing the SciPy Stack for details.

For a quick introduction to ExPy we provide the ExPy Tutorial.
We also recommend the SciPy Lecture Notes for a broader introduction to the scientific Python ecosystem.
